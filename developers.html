    Want to contribute?

    Want to get insider access to Onyx's development? Here are 3 ways you can do so.

    Invite Onyx To Your Server

    Contact Silvia on Discord.

    Firstly, you'll need to let me know of your idea via Discord. Add @Silvia923#9909 and send me your idea.

    Then, if I approve it, either I'll code it, or you can, depending on whether you want to code or not :p

    Join Onyx's Support Server.

    The Onyx Support Server comprises a fun community of developers, server owners, and Discordians who get insider access to all of Onyx's latest features.

    JOIN ONYX's SUPPORT SERVER

    Submit A Pull Request via GitHub

    If you've got some code to add, just submit a pull request via GitHub.

    It'll either get accepted, accepted with some changes, or rejected, since certain commands can't be accepted, such as NSFW commands.
